# Quickstart: Todo Manager

## Prerequisites
- Python 3.10+
- Node.js 18+

## Backend Setup
1. Navigate to `backend/`
2. Create venv: `python -m venv venv`
3. Install: `pip install fastapi uvicorn pydantic`
4. Run: `uvicorn main:app --reload`

## Frontend Setup
1. Navigate to `frontend/`
2. Install: `npm install`
3. Run: `npm run dev`
